This simple tool will allow you to display static and animated maps. It is designed for use with a TV for in person D&D play.

1) Download .zip file
2) Extract
3) Run Index.html
4) How to Use

Load Map: Select a folder.

Pan/Zoom: 'Pan & Zoom' mode.

Fog: 'Edit Fog'. Left Click to reveal, Right Click to hide.

Player View: Open separate window.

Tip: Middle mouse always pans.
