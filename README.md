This contains everything you need to run your app locally. I know absolutely nothing about coding. I used Google AI studio to make a basic map tool when running D&D in person with a TV. It works with both static and animated maps.

Download the .zip file and run the index.html


**AI Generated Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
